# Gen 5 Save Fixtures

- `projectpokemon-base-*-boy.sav` files come from Project Pokemon's "Base Pokemon Save Files (PKHeX compatible)" download set.
- `pkhex-haxorus.pk5` comes from the PKHeX.Core legality test fixtures under `Tests/PKHeX.Core.Tests/Legality/Legal/Generation 5/612 - Haxorus - A1AC5BB800F7.pk5`.

These fixtures cover Black, White, Black 2, and White 2 save containers plus a known-good PK5 entity for encrypted party-data parsing tests.
